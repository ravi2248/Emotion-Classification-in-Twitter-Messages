About the Project

This project is a capstone submission for Mercy University, designed to classify emotions from text data using Natural Language Processing (NLP).

Methodology

The project compares classic machine learning models (like Logistic Regression and SVM) against more advanced deep learning architectures (like LSTMs). After comprehensive training and evaluation, the Bidirectional LSTM was found to be the most effective model, achieving the highest accuracy and the most balanced performance across all six emotion categories.
